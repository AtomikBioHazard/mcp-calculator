# Calculator with MCP Server in JavaScript

A simple calculator that communicates with an MCP server, written in JavaScript.

## Features

- Basic operations: addition, subtraction, multiplication, division
- MCP server communication
- JavaScript-based

## Installation

```bash
git clone <repo_name>
cd <repo_name>
npm install
```

## Usage

To run locally:

- npm run build

Run using the inspector:

- npx @modelcontextprotocol/inspector node build/index.js
